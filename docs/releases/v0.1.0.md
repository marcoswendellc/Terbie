# Terbie v0.1.0 — Foundation

## Visão geral

Esta release consolida a fundação do Terbie como motor analítico conversacional. O projeto passa a ter uma base arquitetural limpa para interpretar perguntas de negócio, organizar conhecimento semântico, compilar intenções analíticas em planos estruturados e executar consultas simples com uma cadeia desacoplada de componentes.

## Entregas incluídas

- FastAPI
- Google Sheets real
- Schema Discovery
- Data Catalog
- Semantic Layer
- Knowledge Layer
- Compiler
- AnalyticalHypothesis
- AnalyticalPlan
- ExecutionPlan
- Logical Query Plan
- Gemini ReasoningProvider
- Executor MVP
- Narrator MVP
- Multi DataSource Foundation
- Documentação arquitetural
- ADRs
- Testes automatizados

## Qualidade

- 70 testes passando
- Ruff sem erros

## Limitações conhecidas

- Executor ainda limitado a operações básicas
- Gemini ainda usado apenas para hipótese analítica
- Sem Runtime analítico completo
- Sem Physical Query Plan
- Sem RAG
- Sem agentes
- Sem MCP
- Sem interface web
